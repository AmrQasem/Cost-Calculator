Support Scripts Description:
############################
This Folder contains DB Scripts that are created to 
support DB at any stage (Deployment, production, testing, ...)

It can contain automation scripts to migrate data,
add data in an automated way, drop db in forced way, etc...


When to Add Scripts to this Folder:
###################################
If you struggled doing something on the DB and you managed
to create/find script to do that thing,
then this script belongs to this Folder
